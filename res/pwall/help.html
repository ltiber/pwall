What is pwall App?

pwall is modern style photo viewer and organizer. It shows the folders of your photos main directory in the left pane and a photowall in the right.

The directories are descending sorted.

The photowall is a scrolling panel of square thumbnails giving a preview of each photo. Loading and unloading of this scrolling panel is made dynamically, viewing page by viewing page like smartphones do. To scroll through your photo library, you can select a folder in the left panel or scroll directly in the photowall. Going down and up will lead you to next or previous directory.

pwall also allows you to have basics management of photos (change dates, copy a group of photos to another directory, delete, share, get GPS data...) but doesn't have edit functions.

The photowall shows the photos in a descending sorted on the file modification date for each directory. If your modification date doesn't represent the taken date, you can use the change dates facilities to get the correct sorting.

Clicking on a photo opens it in fullscreen mode. You can continue your navigation in this mode with left and right keys.

pwall supports only jpg and png files. But for my usage, it was enough.

pwall doesn't import photos in a specific database. It only uses the file system and can handle smoothly 50000 photos with a modern laptop.

You can use pwall as an alternative to Shotwell to browse your photo gallery.

Actions

Main menu

  • Refresh Thumbnails: This process will create thumbnails for all your photos. It can be long!!! The app uses thumbnails to display photo walls and it can take time to create one for each photo. On basic laptop, it can take 7mn to generate thumbnails for 10000 photos. Each thumbnail costs about 10ko and is stored in you home dir in the .pwall subdir. The best thing is to run once at first time or when you renew massively your photos. Updates will be done based on file's last modified date. Nevertheless feel free to cancel this operation if it is not the right moment.The app will create thumbnails on the fly. But the UI might freeze some time to time.
  • Change photos directory gives you the option to specify again the location of your photos library
  • Help gives you access to this current help.

Actions in the directory tree

Note that when you open the app or when you select a new directory to display, the focus leaves the tree panel and goes directly to the photowall. This means that you can scroll directly with keys up and down to catch the photo you need. The drawback is that you never get the focus a long time in the tree panel.

  • Just click on a directory to view its photowall
  • Ctrl-left put the focus on the directory tree and go up and down in the tree with keys. Mind the directory is refreshed 300ms after the new dir selection. So if you move quickly with key up and down to a directory, the app won't load the in between directories in the photowall to save time and be smoother.
  • Ctrl-F to search in the directory tree. It will also hide the searchbar when it is no more needed.
    • Enter the word or part of it and [enter]
    • The first found folder will be selected
    • To go to next result if any, use buttons
  • Right click on a directory to open the context menu
    • "Search Tree" to open the search box
    • "Refresh Tree" to seek for changes in the disk
    • "Rename Folder" open a dialog box to rename it
    • "Move Folder" open a dialog box to select the new parent folder
    • "Delete Folder" deletes the folder if it is empty
    • "New Folder" attaches a new folder to the selected folder

Actions in the photowall

  • Use right, left, top, bottom keys to navigate through the photowall
  • ctrl-left to put the focus on the directory tree and go up and down in the tree with keys
  • Use enter or ctrl-O to open the selected photo in full screen
  • Alt-enter to get the properties of the photo
  • Right click or Context Menu key to open context menu
    • "Refresh" There is no automatic refresh if some changes happen in the file system with other tools. Just click this option to update the photowall.
    • "Open with > Terminal" to open the terminal with the directory of the selected photos. You can run any commands over photos. ctrl-shift-V drops the file names in your terminal from the clipboard
    • "Open with > Google Maps" to get the location of the focused photo if GPS data exists.
    • "Open with > ..." the other actions are the photo viewers or editors found in your machine
    • "Share > Mail" to send the selected photos with your default mail application (Geary, Thunderbird, xdg-email compliant app)
    • "Share > Web Whatsapp" to send the focused photo via whatsapp (when you attach the photo use ctrl-V to get the current photo)
    • "Share > Facebook" to post the focused photo to Facebook (when you attach the photo use ctrl-V to get the current photo)
    • "Properties" show the exif data of the photo. You can copy all the data of this box for further usage.
    • "Copy To" copies the current selection to a chosen directory via a dialog box
    • "Move To" moves the current selection to a chosen directory via a dialog box
    • "Date Operations > Change file's date with Exif's date" updates the file's modification date with the date taken exif data
    • "Date Operations > Entry a new file's date" updates the file's modification date and the date taken exif data with a new date/time

  • Escape key to quit fullscreen mode and return to the photowall
  • Alt-enter to get the properties of the photo

  • Right click or Context Menu key or Ctrl-O to open the context menu
    • "Open with > Terminal" to open the terminal with the directory of the photo and run commands over photo (ctrl-shift-V drops the file name in your terminal)
    • "Open with > Google Maps" to get the location of the current photo if GPS data exists.

    • "Share > Facebook" to post the current photo to Facebook (when you attach the photo use ctrl-V to get the current photo)

Installation comments

  • the pwall cmd is installed in /usr/bin
  • the application resources are installed in /usr/share/pwall
  • the app icon is in /usr/share/icons/hicolor/128x128/apps
  • the .desktop is installed in /usr/share/applications
  • the .ini and the created thumbnails are in ~/.pwall
